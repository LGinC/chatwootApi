# ChatWootApi

English | [简体中文](README.zh-CN.md)

Strongly typed .NET SDK for the Chatwoot REST APIs.

Packages:

- `ChatWootApi`: API interfaces, models, JSON serialization context, and webhook helpers.
- `ChatWootApi.Extensions`: dependency-injection registration for Refit clients, base address configuration, and `api_access_token` handlers.

Install:

```bash
dotnet add package ChatWootApi.Extensions
```

See package READMEs for usage details:

- [ChatWootApi](src/ChatWootApi/README.md)
- [ChatWootApi.Extensions](src/ChatWootApi.Extensions/README.md)
